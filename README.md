১️.What is the difference between var, let, and const?

var, let, এবং const — এই তিনটাই জাভাস্ক্রিপ্টের  ভ্যারিয়েবল ডিক্লেয়ার  করবার জন্য ব্যবহার হয়। কিন্তু এদের কিছু পার্থক্য আছে।

var পুরনো পদ্ধতি,একই ভ্যারিয়েবল আবার ডিক্লেয়ার  করা যায়। অনেক সময় স্কোপ সমস্যা তৈরি করে।

let ইউনিক এবং বেশি ব্যবহার করা হয়,একই ভ্যারিয়েবল আবার ডিক্লেয়ার  করা যায় না। তবপ ভ্যালু 
পরবর্তীতে পরিবর্তন করা যায়।

const এটাও ইউনিক পদ্ধতি।  const  ডিক্লেয়ার  করবার পর ভ্যালু আর পরিবর্তন করা যায় না।
সাধারণত cons ভ্যালু রাখার জন্য ব্যবহার করা হয়।


২️.What is the spread operator (...)?

Spread operator (...) ব্যবহার করা হয় কোন একটা অ্যারের  বা অবজেক্টের সব ইলিমেন্ট একসাথে ছড়িয়ে দেওয়ার জন্য।Spread operator দিয়ে অ্যারে বা অবজেক্ট কপি করা বা নতুন ইলিমেন্ট যোগ করা সহজ হয়।

৩️.What is the difference between map(), filter(), and forEach()?

এই তিনটা ফাংশন  অ্যারের উপর কাজ করার জন্য ইউসড হয়।

map()
অ্যারের  প্রতিটা ইলিমেন্ট নিয়ে নতুন অ্যারে তৈরি করে।


filter()
নির্দিষ্ট কন্ডিশন অনুযায়ী ইলিমেন্ট বেছে নেয়।


forEach()
শুধু প্রতিটি ইলিমেন্টের  উপর লুপ চালায়।
কিন্তু নতুন অ্যারে তৈরি করে না।

৪️.What is an arrow function?
Arrow function হলো জাভাস্ক্রিপ্টের  একটি শর্ট বা ছোটভাবে ফাংশন লেখার পদ্ধতি।Arrow function দিয়ে কম কোডে সহজভাবে ফাংশন  লেখা যায়।

৫️. What are template literals?

Template literals ব্যবহার করা হয় স্ট্রিংয়ের  ভিতরে ভ্যারিয়েবল বা এক্সপ্রেশন বসানোর জন্য।
এটা লিখতে ভ্যাকটিক ব্যবহার করা হয়।Template literals দিয়ে স্ট্রিংয়ের ভিতরে সহজে ভ্যারিয়েবল বসানো যায়।
